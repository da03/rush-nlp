# ProgramAsWeights with agents and coding assistants

- PAW works well as an agent preprocessing layer: parse tool calls, validate outputs, route tasks, classify messages, and repair structured formats before a larger agent acts.
- The website has an Agents page with coding-agent and AI-assistant integration notes: https://programasweights.com/agents
- PAW functions can be chained with regular Python logic - for example, one classifier program routes to another fixer or extractor program.
