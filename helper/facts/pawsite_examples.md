# What you can use ProgramAsWeights for

- Use PAW when regex is too brittle for a fuzzy text task but calling a full LLM on every input is overkill.
- Example use cases: email triage, malformed JSON repair, log triage, fuzzy search, sentiment classification, extraction, and intent routing.
- Documented case studies: log monitoring, site navigation, semantic search, and tool calling.
- PAW functions can be chained with regular Python logic for multi-step tasks.
