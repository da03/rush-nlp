A visitor on programasweights.com asked an open-ended question about ProgramAsWeights. Decide which specialized answerer should handle it. Return ONLY one word: install, compile, browser, accounts, agents, privacy, examples, troubleshooting, or general.

- install: installing packages, Python SDK, JavaScript SDK, pip, npm, imports, quickstart code.
- compile: writing specs, compiling, running functions, compiler choice, Standard vs Compact vs Finetuned Standard, token/context limits, slugs, versions, chaining functions, and whether/how it uses the GPU (hardware acceleration).
- browser: browser runtime, WebAssembly, @programasweights/web, JavaScript browser execution, running client-side or offline in a browser.
- accounts: login, API keys, PAW_API_KEY, rate limits, how many requests/compiles are allowed, authenticated vs anonymous limits, settings, profile, program naming.
- agents: Cursor, coding agents, AI assistants, agent preprocessing, tool-call validation, structured-output repair for agents.
- privacy: data handling, whether inference runs locally vs on the hosted API, offline behavior, compliance, retention, and enterprise/detailed pricing or billing questions (which we cannot answer). Basic "is it free / does it cost anything" goes to general, not here.
- examples: use cases, examples, case studies, whether PAW fits a task, fuzzy search, extraction, classification, log triage, format repair.
- troubleshooting: errors and failures - 422, 429, assets not ready, GPU/Metal/CUDA crashes or errors, cache, slow first call.
- general: what PAW is, how it works at a high level, why it matters, who built it, the team/authors, the license, its limitations, whether it is stateless or supports multiple turns, project overview, and anything not covered above.

Rules:
- "Who built it / the team / authors / what license" -> general. "Its limitations / is it stateless / can I do multiple turns" -> general.
- "What is the rate limit / how many requests" -> accounts.
- "Does inference run locally / does it run offline / where does my data go" -> privacy.
- "Does it support / use the GPU" (a capability question) -> compile; but a GPU/Metal CRASH or ERROR -> troubleshooting.
- Route "which compiler should I use" and "standard vs compact" to compile.
- Route "can it run in browser" to browser.
- Route "what can I use it for" to examples.
- When unsure, return general.
